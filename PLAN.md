# Wargame — План розробки

## Відкриті питання

- [ ] **Бойовий контакт** — як він вирішується? Автоматичний розрахунок з рандомом, чи гравець щось вирішує в момент зіткнення?
- [ ] **Дезінформаційна механіка** — гравець натискає кнопку і витрачає ресурс, чи тонше — через реальні переміщення хибних юнітів?
- [ ] **Туман війни — візуалізація** — як гравець бачить "застарілу" розвідку? Прозорість? Поступове зникнення позицій?
- [ ] **MVP scope** — що з 7 пунктів найризикованіший технічно?
- [x] **Режим гри** — реальний час з паузою і контролем швидкості

### Юніти гравця
- [x] Ієрархія: Бригада (HQ) → Батальйон (HQ) → Роти (пряме керування гравця)
- [x] Різноманіття рот дає тактичну глибину: штурмові, танкові, розвідувальні, протитанкові, мінометні, інженерні
- [x] Спецура (Кракен, ССО) — окремі групи, швидкі, невидимі у тумані війни
- [x] ТРО — утримання позицій, слабкі в атаці
- [ ] Скільки рот від кожного типу бригади? (визначає загальну кількість юнітів)
- [ ] Туман війни — видимість на рівні рот чи батальйонів?
- [ ] Підкріплення по ходу гри чи всі юніти з початку?

### Структура бригада → батальйон (когезія)
- [x] Бригади складаються з батальйонів як окремих юнітів на карті; бригада сприймається як єдине ціле
- [x] Когезія: якщо батальйони далеко від HQ або один від одного — readiness знижується
- [ ] HQ бригади — окремий юніт що рухається вручну, чи переміщується автоматично?
- [x] Батальйон — логічний контейнер без фізичного HQ на карті; групує роти для когезії і readiness
- [ ] Скільки батальйонів у бригаді? 2–3 чи більше?
- [ ] Когезія впливає тільки на readiness, чи також на доступні тактики і туман війни?

- [x] Ініціалізація проєкту: Vite + React

- [x] Вивести карту центровану на зону операції (Балаклія / Куп'янськ / Ізюм)
- [x] Налаштувати bounds, zoom, стиль карти (satellite)
- [x] Маска затемнення за межами зони операції
- [x] Flat-top гексова сітка поверх карти

### Фаза 2 — Юніти на карті ✓
- [x] Визначити модель даних юніта (Brigade → Battalion → Company)
- [x] Розмістити стартові позиції (80, 95, 25 бригади)
- [x] Відрендерити юніти на гексах (NATO символи + шеврони бригад)
- [x] Вибір юніта кліком
- [x] Анімоване переміщення між гексами
- [x] Zustand store

### Фаза 3 — Панель юніта

- [ ] Відображення шеврону бригади в панелі

### Фаза 4 — Туман війни
